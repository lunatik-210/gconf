GitHub Models полностью отключат 30 июля для всех пользователей. Исчезнут playground, каталог моделей, inference API и BYOK endpoints. Первый тестовый brownout прошёл 16 июля, следующий назначен на 23-е: запросы временно будут возвращать ошибки.

Если GitHub Models есть в вашем контуре, не ждите финального shutdown. До 23 июля найдите все вызовы, назначьте fallback и прогоните через него один реальный запрос.

Не путать с GitHub Copilot: речь именно о сервисе GitHub Models.
