GitHub добавил AI security detections прямо в pull request: находки запускаются при открытии или обновлении PR, помечаются `AI` и не блокируют merge.

Пока это public preview для GitHub Code Security: нужны CodeQL default setup, разрешение Enterprise, лицензия Copilot и AI credits. Независимых данных о точности нет.

На одном некритичном репозитории сравните обычный review с новым слоем: сколько находок подтвердилось и сколько оказалось шумом?

Источник: [GitHub Changelog](https://github.blog/changelog/2026-07-14-code-scanning-shows-ai-security-detections-on-pull-requests/), 14 июля 2026.
