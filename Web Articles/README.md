# Web Articles

Локальная подборка первичных веб-источников для исследования GCONF.

- Срез собран: `2026-07-17`.
- В корпус включены только публикации на официальных сайтах AI-лабораторий.
- Одна папка соответствует одному источнику.
- Каждый snapshot содержит полный нормализованный текст статьи в `article.md`,
  проверяемые метаданные и checksum в `metadata.json`, а также отдельный
  редакционный пересказ в `research-note.md`.
- Числа и сравнения моделей здесь являются заявлениями соответствующих лабораторий, пока не подтверждены независимым источником.

Структура пакета:

```text
<Lab>/<article-id>/<collected-date>/
├── metadata.json
├── article.md
└── research-note.md
```

Тематическая карта и логика отбора находятся в
[`research/web_articles_three_topics_2026-07-17.md`](../research/web_articles_three_topics_2026-07-17.md).
